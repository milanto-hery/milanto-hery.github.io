Bioacoustic Classification Pipeline

Developed an automated workflow for biodiversity monitoring using Passive Acoustic Monitoring (PAM). The system integrates signal preprocessing and Convolutional Neural Networks (CNNs) to classify wildlife species from large-scale ecological recordings. This project focuses on scalability and accuracy in real-world environmental conditions, providing a robust solution for long-term ecological data collection and analysis.

Real-time Ecological Surveillance

Implemented a robust frame-based object detection system using the YOLO architecture. Optimized for continuous video streams, the pipeline enables real-time inference and visualization for automated ecological monitoring. The system is designed to handle high-throughput data and complex backgrounds, making it suitable for monitoring insects and other small organisms in their natural habitats.

Edge AI for Wildlife Alerts

Engineered a low-cost Edge AI system for avian detection deployed on Raspberry Pi hardware. The solution performs on-device inference and transmits real-time alerts via Telegram, ideal for low-connectivity environments. By processing data locally, the system reduces power consumption and bandwidth requirements, enabling long-term deployment in remote areas for critical wildlife surveillance.

Primate Analysis Interface

Created an interactive Streamlit application for analyzing wildlife imagery, specifically focusing on lemur detection. The platform enables users to deploy trained deep learning models through an intuitive web interface for rapid ecological insights. The interface supports various image formats and provides real-time feedback on model predictions, empowering researchers to efficiently process large datasets and focus on conservation priorities.
